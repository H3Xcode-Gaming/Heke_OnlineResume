Inventory System

This is a Basic Inventory system where you can add, edit, delete items. you can load personal inventory files and save the file to the InventoryList.txt file. The code consists of an Item class, Node class and Inventory class which has a built in Doubly linked list and quick sort via price or quantity. You can load multiple files or add new items to the current file to either the top of the list or bottom.

Project Overview

This is a very basic inventory system where you can Add, Edit, Delete, Display, Load Inventory file from typing the file path in directly and Save the file to Inventory.txt, a default file inside SavedInventory file. You can sort the inventory by price or quantity. This uses a QuickSort algorithm built into the inventory class for both Ascending and Descending order. the inventory class also has a build in Doubly linked list utilising the Node class and Items class to create each item giving you an opportunity to add items to either the end or the start of the list. You can search any item you would like to edit or delete via name. once finished you can save the current inventory which will save all the items to the inventory.txt and remove them from the current list in order of Name, Type, Price and Quantity.

This was created using Visual Studios 2019 and has no dependacies outside 3 header files and 2 cpp files. This can also still be used without the SavedInventory file but the save function may not work and the load function may only work if you type in the file path.
