Live Link

Project Overview

This game is a remake of the classic game Defenders. I have adapted the game from Fighter Jet VS Aliens to Ninja VS Bandits. the layout of the game is similar with the minimap, supplies being taken (Previously People) and the standard side to side movement. However, I have made this game as a Ninja running, jumping and swinging along to stop the bandits taking any supplies. You aim with the mouse cursor and click to throw knives (kunai) at the bandits to kill them. You gather points as you take the bandits out, but the kunai you throw also cost points to stop over use of the throwing mechanic.

This game was created using GameMaker.
